# 📒 クリプト損益台帳

暗号資産取引の確定申告をサポートするWebアプリです。

## 機能

- **損益記録** — USD / JPY 建て取引の利益・損失を入力
- **経費記録** — ガス代・手数料などの経費管理
- **USDC円転** — 円転履歴と為替損益の自動計算
- **税金計算** — 課税所得・推定税額の試算（固定税率 / 累進税率テーブル）
- **為替レート自動取得** — ExchangeRate-API（市場中間値）を利用
- **CSV エクスポート / インポート** — データのバックアップと復元
- **ダークモード対応**

## データ保存

すべてのデータはブラウザの `localStorage` にのみ保存されます。外部サーバーへの送信は一切ありません。

## 為替レートの法的根拠

所得税基本通達 法第57条の3《外貨建取引の換算》関係 3-2 に基づき、ExchangeRate-API の市場中間値を継続使用しています。
